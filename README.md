# MiniArm

miniarm_autoxy 是一个很简单的测试程序，4个舵机分别旋转一下。其他的程序还在持续更新。


拿着手机不知所措的同学，请按上面的 View code 就可以看到程序和说明书了，部分手机不能在线阅读pdf，可以下载后再看。

如果还是不会，请Google一下看看GitHub咋用，或者放弃后用web版编译器编译
